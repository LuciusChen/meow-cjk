* meow-cjk

CJK (Chinese, Japanese, Korean) word segmentation support for [[https://github.com/meow-edit/meow][Meow]] modal editing.

** Features

- Intelligent CJK word segmentation
- Seamless integration with Meow
- Fast and accurate word boundaries
- Cross-platform (macOS/Linux/Windows)

** Requirements

- Emacs 27.1+
- [[https://github.com/meow-edit/meow][Meow]] 1.0+
- [[https://github.com/roife/emt][EMT]] 2.0.0+

** Installation

*** straight.el

#+begin_src elisp
(use-package emt
  :straight (:host github :repo "roife/emt")
  :config (emt-mode 1))

(use-package meow-cjk
  :straight (:host github :repo "LuciusChen/meow-cjk")
  :after meow
  :config (meow-cjk-mode 1))
#+end_src

*** package.el

#+begin_src elisp
;; Install EMT
(unless (package-installed-p 'emt)
  (package-vc-install "https://github.com/roife/emt"))

;; Clone and load meow-cjk
(add-to-list 'load-path "~/.emacs.d/site-lisp/meow-cjk")
(require 'meow-cjk)
(meow-cjk-mode 1)
#+end_src

** Usage

Enable ~meow-cjk-mode~ to enhance these Meow commands with CJK support:

| User command              | Advised internal       | Description                                          |
|---------------------------+------------------------+------------------------------------------------------|
| ~meow-mark-word~          | ~meow-mark-thing~      | Mark CJK word under cursor (expandable selection)    |
| ~meow-next-word~          | ~meow-next-thing~      | Select to end of next CJK word                       |
| ~meow-back-word~          | ~meow-next-thing~      | Select to beginning of previous CJK word             |
| ~meow-kill-word~          | ~meow-kill-word~       | Kill forward to CJK word boundary (~M-d~ in INSERT)  |
| ~meow-backward-kill-word~ | ~meow-kill-word~       | Kill backward to CJK word boundary (~M-DEL~ in INSERT) |

** Example

#+begin_example
Text: 测试文本就是我
      ^
      cursor here

meow-mark-word → selects "测试"
meow-next-word → moves to "文本"
meow-kill-word → kills "测试"
#+end_example

- Backends

** EMT (cross-platform via ewt-rs)

Uses ewt-rs, which delegates to the platform's native NLP tokenizer:

- macOS/Linux: ICU tokenizer
- Windows: WinRT or ICU tokenizer

** Planned

- Jieba (cross-platform, Python-based)

- Configuration

#+begin_src elisp
;; Basic
(use-package meow-cjk
  :after meow
  :config (meow-cjk-mode 1))

;; With performance tuning
(use-package meow-cjk
  :after meow
  :custom
  (emt-use-cache t)
  (emt-cache-lru-size 200)
  :config (meow-cjk-mode 1))
#+end_src

- Troubleshooting

** EMT module not found

#+begin_src elisp
M-x emt-download-module  ; Download pre-built
M-x emt-compile-module   ; Or compile from source
#+end_src

** Performance issues

Increase cache:

#+begin_src elisp
(setq emt-cache-lru-size 200)
#+end_src
